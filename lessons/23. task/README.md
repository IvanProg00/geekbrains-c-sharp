# Task 23

Set a one-dimensional array of 123 random numbers. Find the number of array
elements whose values lie in the segment `[10, 99]`.

## Example

Example for an array of 5, not 123 elements. In your solution, do for 123.

- `[5, 18, 123, 6, 2]` -> 1
- `[1, 2, 3, 6, 2]` -> 0
- `[10, 11, 12, 13, 14]` -> 5
